# Privates 3D-Haus Walkthrough

Dieses Repository ist privat und enthält eine begehbare 3D-Haus-Web-App.

## Wichtig zur Privatsphäre

Der Code liegt privat in deinem GitHub-Repository. Damit die App aber wirklich online im Browser nur für dich sichtbar ist, braucht sie ein Hosting mit Login-/Access-Schutz, zum Beispiel Cloudflare Pages + Cloudflare Access oder eine private GitHub-Codespaces-Vorschau.

GitHub Pages ist zwar eine einfache Hosting-Möglichkeit für statische HTML/CSS/JS-Dateien, aber GitHub Pages selbst ist ein statischer Website-Host. Ein echter Zugriffsschutz nur für deinen Account ist damit nicht die einfachste sichere Variante.

## Einfachste private Nutzung ohne Download

1. Öffne dieses Repository auf GitHub.
2. Starte einen Codespace.
3. Öffne `index.html` über die Browser-Vorschau / Live Server.

Die App läuft dann online in deinem Browser, aber der Zugriff bleibt an dein GitHub-Konto gebunden.

## Steuerung

- WASD: laufen
- Maus: umsehen
- Shift: schneller laufen
- R: Dach/Decke ein-/ausblenden
- H: Hilfe ein-/ausblenden

Auf dem Handy:

- linke Bildschirmhälfte ziehen: laufen
- rechte Bildschirmhälfte ziehen: umsehen
